# 咖面/会面锚点卡模板

> 用途：非正式面试/咖啡会面前输出，A4一页，打印折叠放口袋或存手机备忘录
> 制作方法：从复盘文档提取核心信息，浓缩为8个模块，用emoji做视觉锚点

## 模板结构

```
┌──────────────────────────────────────────────────────┐
│            {公司} 咖面锚点卡 · {日期}                │
│     {时间} {地点} 对面：{姓名}（{身份}）             │
├──────────────────────────────────────────────────────┤
│                                                      │
│  ☕ 这局本质                                          │
│  不是{什么}。是{什么}。{一句话定调}                   │
│                                                      │
│  🎯 核心定位（一句话记住）                            │
│  {用户的三句话定位}                                   │
│                                                      │
│  📍 开场怎么接（他主导，你别抢）                      │
│  "说说你自己" → {30秒回答}                           │
│  "你怎么看我们" → {1-2句真实感受}                     │
│                                                      │
│  ⚓ 三个柱子（他问"你能干嘛"时）                      │
│  ① {能力1} — {一句话证明}                            │
│  ② {能力2} — {一句话证明}                            │
│  ③ {能力3} — {一句话证明}                            │
│                                                      │
│  💰 关于薪资                                         │
│  {一句话，不主动开价}                                 │
│                                                      │
│  📝 可以主动抛的（创造深度）                          │
│  □ {洞察1}                                           │
│  □ {洞察2}                                           │
│  □ {洞察3（如"我知道自己适合做什么"）}               │
│                                                      │
│  🚩 现场风险信号                                     │
│  {信号1} → {应对}                                    │
│  {信号2} → {应对}                                    │
│                                                      │
│  👋 收尾+后续                                        │
│  主动收：{一句话}                                     │
│  当天→{感谢}  2-3天→{补充}  1周→{温和跟}            │
│  红线：不催结果                                       │
│                                                      │
│  📎 完整复盘 → {复盘文件路径}                         │
└──────────────────────────────────────────────────────┘
```

## 示例（STW赏游科技 2026-07-08）：

```
┌──────────────────────────────────────────────────────┐
│  STW 咖面锚点卡 · 7/8 周三 09:30                     │
│  Cyberport 3 Starbucks 对面：Adam+Emily               │
├──────────────────────────────────────────────────────┤
│  ☕ 这局本质                                          │
│  不是面试。Adam被Emily叫来"看一眼"。他判断：          │
│  这人靠谱吗？能共事吗？要多少钱？                     │
│                                                      │
│  🎯 核心定位                                          │
│  "不是AI工程师，不是创意人。是懂中国、懂小老板、      │
│   能搭体系、能落地的那个人。"                          │
│                                                      │
│  ⚓ 三个柱子                                          │
│  ① 培训体系 — Attention+Memory，搭过认证体系         │
│  ② AI落地 — RAG/智能体/本地部署都做过                │
│  ③ 从0到1 — 搭B2B管道、2个月建3类门店                │
│                                                      │
│  💰 薪资：更在意能不能做出东西来，灵活谈              │
│                                                      │
│  👋 收尾："回去消化一下，随时补充"                    │
└──────────────────────────────────────────────────────┘
```

## 使用原则

- V2推荐版式：方框线用 `┌─┐│└─┘` 字符打印
- 每个模块不超过3行
- emoji作为视觉锚点：☕🎯📍⚓💰📝🚩👋
- A4纸打印对折两次放入口袋，或手机备忘录截图
- 完整复盘文档保留在同一目录下，锚点卡只做提取
