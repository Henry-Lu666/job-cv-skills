# 人格小票 · Personality Receipt

> 根据上传的自我介绍或聊天记录，生成一张热敏小票风格的人格画像。
> 内容：核心类型 / 性格特质 / 能量来源 / 决策习惯 / 压力自救方案 / 给AI的使用说明。
> 相当于 "这份文件是AI理解这个人的快速入口"。

---

## 使用方式

用户提供自我介绍或聊天记录，Agent 分析后输出以下格式的 HTML 热敏小票。

## 分析维度

### 1. 核心类型
从上传材料中识别最突出的类型标签，附一句话注脚。
格式如：`身弱·借印` / `执行者·框架型` / `创意者·发散型`

### 2. 性格特质（4-6条）
每条特质用 {{label}} — {{一句话描述}} 格式。

### 3. 能量来源（2-3条）
什么场景下这个人的表现最好？什么情景最消耗ta？

### 4. 决策习惯
主要决策模式 + 潜在盲区。

### 5. 压力自救方案（3-5条）
当这个人卡住/焦虑/跑偏时，什么方法最有效？

### 6. 给AI的使用说明
一句话概括——怎么跟这个人沟通最顺畅。
加3个tag：如 `先说人话` `给修正机会` `别让ta空想`

## 设计规范

- 风格：复古热敏小票，黄底/金边/等宽字体
- 配色：背景 #fffef5，主线 #d4a843，标题深蓝 #001f5c
- 字体：英文标题 Playfair Display，正文 Inter，代码 DM Mono
- 尺寸：宽 380px，居中
- 边框：2px solid #d4a843 + 4px 圆角 + 8px 阴影偏移
- 顶部/底部的虚线装饰线（热敏纸撕口效果）
